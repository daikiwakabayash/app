# 3D 筋肉・骨格ビューア（整体・研修向けデモ版）

整体店舗での施術説明や、スタッフ研修での解剖学習を想定した、ブラウザだけで動く
**3D 解剖ビューア**のデモです。マウス/タッチで人体を回転させ、筋肉をクリックすると
**起始・停止・支配神経・作用**と、施術視点の「整体メモ」を日本語で表示します。

| 前面 | 背面 |
|---|---|
| 全身の筋を立体的に確認 | 背部・殿部の筋を確認 |

## 特長

- **3D 表示**：ドラッグで回転、ホイール/ピンチで拡大縮小。前面/背面ワンタッチ切替。
- **クリックで詳細**：筋を選ぶとハイライトされ、詳細カードを表示。
- **日本語で統一**：UI・解説はすべて日本語（ラテン語名のみ学術参照として併記）。
- **領域別フィルタ**：頸部／肩・上肢／背部／体幹／殿部・下肢で絞り込み。
- **検索**：筋肉名・作用・神経名で横断検索（例「外転」「坐骨神経」）。
- **骨格のみ表示**：筋肉の表示/非表示を切替え、骨格構造を確認。
- **オフライン動作**：Three.js を同梱。インターネット接続なしでも起動可能（店舗 PC・タブレット向け）。

## 解剖データの根拠

収録している起始・停止・支配神経・作用は、標準的な解剖学の教科書・参考書
（*Gray's Anatomy*、*Netter 解剖学アトラス*、Kendall『筋：機能とテスト』など）の
記載に基づいて整理しています。研修教材として用いる際は、最終確認に成書をご参照ください。

### 収録筋（デモ版・18 筋）

頸部：頭板状筋／肩甲挙筋／胸鎖乳突筋
背部：僧帽筋／広背筋／脊柱起立筋／菱形筋
肩・上肢：三角筋
体幹：大胸筋／腹直筋／外腹斜筋／腰方形筋
殿部・下肢：大殿筋／中殿筋／腸腰筋／大腿四頭筋／ハムストリングス／腓腹筋

> 整体で着目されやすい姿勢・腰痛・肩こり関連の筋を優先して収録しています。

## 使い方（ローカル起動）

ES モジュールを使うため、ファイルを直接開くのではなく簡易サーバ経由で開いてください。

```bash
# 例：Python が入っている場合
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

GitHub Pages 等の静的ホスティングにそのまま配置しても動作します。

## 技術構成

```
index.html            … 画面構成・UI
css/styles.css        … ダークテーマのUI
js/anatomyData.js     … 解剖データ（骨格ランドマーク＋筋データ）
js/model.js           … 骨格・筋の3Dモデル生成
js/app.js             … シーン構築・操作・UI連携
vendor/three/         … Three.js 本体（同梱・オフライン対応）
```

3D 表現は Three.js による手続き的生成です。各筋を**筋束（fascicle）の集合**として、
起始から停止へ走る線維方向に沿って生成し、紡錘形の筋腹・淡色の腱・線維の陰影を
再現することで、解剖アトラスに近い見え方にしています（広幅筋は起始から停止へ
扇状に収束）。解剖学的な位置関係（起始・停止）を優先したモデルであり、
さらに写実的な筋形態の再現は今後の拡張ポイントです。
